PYTHON VERSION
PoopDisplay class API has been tested as working in Python 3

PYTHON MODULES
The following Python modules are used by the PoopDisplay class:
* pygame
* threading
* time
* socket
* sys
* re

SCRATCH VERSION
Only works with Scratch 1.4 as it uses the remote sensors protocol:
https://en.scratch-wiki.info/wiki/Remote_Sensors_Protocol
