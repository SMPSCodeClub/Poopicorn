PoopDisplay:
A Display Interface Class-Based Scratch 1.4 API for use in Python 3


SCRATCH-BASED TEST HARNESS

PoopDisplay.py - the class file
test_example.py - example code that runs through all API functions
TestDisplay.sb - sample Scratch to show the effects of test_example.py


TESTING THE DISPLAY INTERFACE - SCRATCH

1. place everything in the same directory 
2. Load TestDisplay.sb into Scratch 1.4
3. Make sure the remote sensors are ENABLED
4. Click on GREEN FLAG
5. run test_example.py
